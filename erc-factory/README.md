# ERC Factory

ERC Create Factory Contract

## Setup

```
PRIVATE_KEY=
ETHERSCAN_KEY=
DEV_ALCHEMY_KEY=
PROD_ALCHEMY_KEY=
ADMIN_ADDRESS=
```

## Get Started

### Ether

```
npm run deploy:mainnet
npm run deploy:testnet
```

### Polygon

```
npm run deploy:polygon
npm run deploy:mumbai
```

## Get Test

```
npm run test
```

## Get Verify

```
npm run compile
npx hardhat verify --network testnet ${contract address}
```
