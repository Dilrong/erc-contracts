# ERC721 Market

## Setup

```
PRIVATE_KEY=
ETHERSCAN_KEY=
DEV_ALCHEMY_KEY=
PROD_ALCHEMY_KEY=
ADMIN_ADDRESS=
TREASURY_ADDRESS=
```

## Get Started

```
npm run deploy:testnet
```

## Get Test

```
npm run test
```

## Get Verify

```
npm run compile
npx hardhat verify --network mainnet ${contract address}
npx hardhat verify --network testnet ${contract address}
```
